# Sound Card Measurement System

Engineering thesis project focused on the development of a desktop application that uses a computer sound card as a measurement sensor.

The project includes signal generation, spectrum analysis (FFT), impedance measurement, and audio signal processing experiments.

## Structure
- src/ – main implementation
- experiments/ – experimental scripts
- legacy/ – outdated or non-working code
